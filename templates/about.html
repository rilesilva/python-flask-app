Instrucciones de Despliegue en EasyPanel
1. Crear Contenedor en EasyPanel
  1. Accede a tu panel de EasyPanel
  2. Ve a la sección "Contenedores"
  3. Haz clic en "Crear Contenedor"
  4. Selecciona "Docker Compose" o "Docker Image"
  5. Configura el puerto 5000
2. Subir Código
  1. Comprime todos los archivos del proyecto
  2. Sube el archivo ZIP a EasyPanel
  3. O usa Git para clonar el repositorio
  4. Configura las variables de entorno si es necesario
Importante: Asegúrate de que el puerto 5000 esté configurado correctamente en EasyPanel y que las variables de entorno estén definidas.

Variables de Entorno
  • PORT - Puerto de la aplicación (default: 5000)
  • FLASK_ENV - Entorno Flask (development/production)
  • SECRET_KEY - Clave secreta para sesiones
